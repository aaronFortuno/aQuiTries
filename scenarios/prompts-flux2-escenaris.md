# Prompts FLUX 2 — Escenaris de fons (sense personatges)

Cada prompt genera una imatge de fons en pixel art per a una situació del joc.
**Cap humà ni personatge visible** — només l'escenari i la situació.
Format horitzontal/panoràmic per servir de fons de pantalla del joc.

---

## Prompt maestro (base comuna)

```text
[SCENE DESCRIPTION HERE]

No humans, no characters, no people visible anywhere in the scene. Empty environment only.

Art style: pixel art aesthetic inspired by classic video game backgrounds. Visible blocky pixels, limited color palette (no gradients), clean hard edges, no anti-aliasing, no soft shading, no glow effects. Use 3–5 tones per color (light, mid, dark). The image should look like a high-resolution pixel art scene — large visible pixels as a stylistic choice, not a low-resolution image.

Lighting: consistent natural lighting appropriate to the scene, with clear but minimal highlights and shadows.

Composition: wide horizontal format (landscape orientation, roughly 16:9), suitable for a game background, with visual space in the center or sides where character sprites could be placed on top. The scene should feel like a stage or diorama.

Output: high-resolution image in landscape orientation. Full scene visible, no cropping.
```
